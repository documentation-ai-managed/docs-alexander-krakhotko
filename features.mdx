## Document Creation and Editing Tools

You create new documents directly from the dashboard. Use the built-in editor supporting MDX syntax, including components like `<Callout>` and code blocks.

<Steps>
  <Step title="Create a New Document" icon="plus">
    Click the `{New Document}` button in your project sidebar.
  </Step>
  <Step title="Edit with Preview" icon="eye">
    Write in the split-view editor. Preview renders live as you type.
  </Step>
  <Step title="Add Components" icon="code">
    Insert JSX components via the toolbar for interactive docs.
  </Step>
  <Step title="Save and Publish" icon="save">
    Auto-save enables. Hit `{Publish}` to make live.
  </Step>
</Steps>

For example, embed a code snippet:

````mdx
```javascript
// Realistic user auth example
const login = async (email, password) => {
  const response = await fetch('https://api.example.com/auth/login', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ email, password })
  });
  return response.json();
};
```
````

<Callout kind="tip">
  Enable `{rich-editing}` mode for drag-and-drop images and auto-formatting.
</Callout>

## Project Organization and Folders

Organize docs hierarchically. Create projects for apps or teams, then nest folders for guides, APIs, and changelogs.

| Feature | Description | Use Case |
|---------|-------------|----------|
| Projects | Top-level containers | Separate docs for web/app projects |
| Folders | Nested organization | Group `/quickstart`, `/api-reference` |
| Tags | Cross-folder labeling | Mark `{wip}`, `{published}` |

Drag files between folders via the file tree. Folders sync across collaborators.

## Collaboration and Sharing Options

Invite team members with role-based access. Share public links or embed pages.

<Tabs>
  <Tab title="Team Invite" icon="mail">
    Send invites via email. Assign `{editor}`, `{viewer}`, or `{admin}` roles.

    ```bash
    # Admin command-line invite (CLI tool)
    alexander invite user@example.com --role=editor --project=my-docs
    ```
  </Tab>
  <Tab title="Public Sharing" icon="globe">
    Generate shareable links. Set expiration or password protection.
  </Tab>
  <Tab title="Embed" icon="code">
    Copy embed code for sites.

    <CodeGroup tabs="HTML,React">
    ````html
    <iframe src="https://docs.example.com/my-project/embed" width="100%" height="600"></iframe>
    ````
    ````jsx
    <iframe src="https://docs.example.com/my-project/embed" style={{width: '100%', height: '600px'}} />
    ````
    </CodeGroup>
  </Tab>
</Tabs>

<Expandable title="Advanced Search Syntax" default-open="false">

      B --> C[Filter by Project/Folder]

Navigate via auto-generated sidebar. Breadcrumbs track your location.
